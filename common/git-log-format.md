- Formating options for git log

`git log --pretty=format:'<options>'`

- Commit info

`{{ %H }} Commit hash`
`{{ %h }} Abbreviated commit hash`

- Tree
`{{ %T }} Tree hashes`
`{{ %t }} Abbreviated tree hash`

- Parents
`{{ %P }} Parent hashes`
`{{ %p }} Abbreviated parent hashes`

- Author

`{{ %an }} Name`
`{{ %ae }} Email`
`{{ %ad }} date (format respects the --date=option)`
`{{ %ar }} date, relative`

- Body and Subject

`{{ %s }} Subject`
`{{ %B }} Message`
